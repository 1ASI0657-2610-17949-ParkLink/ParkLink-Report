# ParkLink Keynote Narrative Plan

Audience: docente y evaluadores del curso.

Objective: presentar ParkLink con foco en las correcciones solicitadas: segmentos claros, matriz de tareas, análisis de tarifas, riesgos, TS/RNF y arquitectura.

Slide list: carátula, equipo, problema y segmentos, Lean UX y entrevistas, competidores, tarifas, user task/as-is/to-be, historias, impacto/riesgos, arquitectura, cierre.

Visual system: fondo claro, verde ParkLink como color principal, azul para arquitectura/segmento empresario y ámbar para pricing/decisiones.

Editability plan: todos los textos clave, tablas simplificadas, tarjetas y diagramas se construyen como objetos editables de PowerPoint.
